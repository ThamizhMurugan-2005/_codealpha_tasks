
# AI Chatbot

## Project Overview
The AI Chatbot is a Java-based application developed as part of the **CodeAlpha Cloud Computing Internship**. This project simulates a chatbot that interacts with users by responding to simple text-based queries. It demonstrates the basic concepts of Artificial Intelligence and Natural Language Processing (NLP) using predefined responses.

## Objective
- Develop a chatbot that interacts with users.
- Provide instant responses to user queries.
- Demonstrate the basic concept of AI-based conversation.
- Improve user experience through automated communication.

## Features
- Accepts user input.
- Responds to predefined questions.
- Simple console-based chatbot.
- Easy to use and understand.
- Demonstrates basic AI conversation logic.

## Technologies Used
- Java
- Scanner Class
- Conditional Statements (if-else)
- Basic AI Chatbot Logic

## How It Works
1. The chatbot starts and waits for user input.
2. The user enters a question or greeting.
3. The chatbot compares the input with predefined responses.
4. If a matching response is found, it displays the answer.
5. Otherwise, it displays a default message.
